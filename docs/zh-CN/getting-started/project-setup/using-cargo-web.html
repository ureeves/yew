使用 cargo-web

Cargo web 是一个用来构建 web 客户端应用的 Cargo 子命令，它让构建和部署 web 应用变得非常的简单。它同样也是唯一一个支持生成 Emscripten 目标代码的工具链。点击这里了解更多。

安装

cargo install cargo-web

构建

cargo web build

运行

cargo web start

支持生成的目标代码

  • wasm32-unknown-unknown
  • wasm32-unknown-emscripten
  • asmjs-unknown-emscripten

{% hint style="info" %} 对于 *-emscripten 的目标代码， 你需要安装 Emscripten SDK。 {% endhint %}
